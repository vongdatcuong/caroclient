A ReactJS wep application for playing Tic-Tac-Toe (a board game) online .
- Client site: users can create or find a room to join, then click ready and play game. They can also withdraw
from a match, reconnect to an ongoing game when being disconnected, or watch an ongoing game.
- Server site: Provide socket server which allows users to interact with each other in real time, and API to
make changes to database
Tech stack: Javascript, ReactJs, Material-UI, NodeJS, ExpressJS Framework, MongoDB,Mongoose,Socket.IO, Git.
